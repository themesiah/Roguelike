El objetivo de este prototipo es mostrar una primera versión del primer nivel con generación aleatoria. Actualmente hay 18 salas diferentes, 25 minions diferentes y 20 equipos diferentes.

Por cuestiones de eficiencia en el prototipiado, esta vez no hay instrucciones ingame.

Controles (mapa):
- Movimiento: WASD, teclas de dirección, joystick izquierdo
- Mirar hacia abajo: S, dirección abajo, joystick izquierdo hacia abajo
- Salto: Espacio, A(xbox), X(ps)
- Dejarse caer de una plataforma: S/dirección abajo + Salto, joystick izquierdo hacia abajo + Salto
- Interactuar: Z, Y(xbox), Triángulo(ps)
- Interactuar (equipo): X, X(xbox), Cuadrado(ps)
- Abrir menú de pausa: Escape, Start, Start
- Abrir menú de información: Tab, Select, Select

Controles (ui):
- Moverte entre opciones: Ratón, Teclas de dirección, joystick izquierdo
- Seleccionar: Ratón izquierdo, Z, Enter, A (xbox), X (ps)

Controles (batalla):
- Usar habilidad: WASD, ABXY, X cuadrado redonda triángulo
- Detener tiempo: mantener F, mantener RT, mantener R2
- Shuffle: R, LB, L1
- Ultimate: Q, RB, R1
- Cambiar objetivo (marcado por flechita): Teclas de dirección, joystick izquierdo


Cambios respecto a prototipos anteriores:
- El sistema de batalla es el mismo, pero hay más variedad de opciones respecto a minions y equipos.
- Muchas salas nuevas! El prototipo es autocontenido, así que en una sola run se pueden explorar las 9 salas.
- Arte nuevo para las salas.
- En este prototipo NO hay boss final.
